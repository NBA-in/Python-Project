# Modern Marks Calculator 🎓

## Project Overview
[cite_start]This project is a Python-based desktop application developed as an Individual Mini-Project for the Programming in Python course[cite: 3]. It provides a modern Graphical User Interface (GUI) to calculate a student's total marks, percentage, and corresponding letter grade based on their input scores. 

[cite_start]To meet the requirement of using a library beyond the core syllabus[cite: 21], this project utilizes **CustomTkinter** to deliver a sleek, dark-mode-compatible interface that significantly improves upon the standard Tkinter library.

## ✨ Features
* **Modern UI:** Built with CustomTkinter for smooth corners, hover effects, and automatic system light/dark mode detection.
* **Robust Input Validation:** Prevents application crashes by catching non-numeric inputs and explicitly warning users if they enter numbers outside the logical 0–100 range.
* **Dynamic Color-Coding:** The final letter grade changes color based on performance (e.g., Green for A, Red for F).
* [cite_start]**Clear Workflow:** Demonstrates a stable and simple Input-Process-Output workflow[cite: 24, 25, 26, 27, 28].
